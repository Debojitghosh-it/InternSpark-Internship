## ⚙️ How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/Debojitghosh-it/InternSpark-Internship.git
   cd InternSpark-Internship
   ```
2. Navigate into any task folder
   ```bash
   cd Task-1- Model Deployment-API & Contenerization
   ```
3. Install dependencies (if a `requirements.txt` is present)
   ```bash
   pip install -r requirements.txt
   ```
4. Run the script
   ```bash
   python main.py
   ```
